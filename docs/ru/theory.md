# Теория Ternary Gradient Compression (TGC)

Ternary Gradient Compression (TGC) — это метод сжатия данных, который квантует значения градиента в три уровня: $\{-1, 0, +1\}$. В отличие от 1-битного SignSGD (который передает только знаки), TGC вводит "зону нечувствительности" для малых значений, что значительно повышает устойчивость модели к шуму.

## Математическая модель

Пусть $g$ — вектор градиентов. Оператор квантования $Q(g)$ определяется как:

$$Q(g_i) = \begin{cases} +s & \text{if } g_i > \gamma \\ 0 & \text{if } |g_i| \le \gamma \\ -s & \text{if } g_i < -\gamma \end{cases}$$

Где:
- $s$ — масштабный коэффициент (обычно среднее абсолютное значение градиентов выше порога).
- $\gamma$ — пороговое значение (threshold).

### 1. Выбор порога (Thresholding)
Статический порог может быть неэффективен. Оптимальный порог $\gamma$ часто выбирается как доля от максимального градиента или на основе стандартного отклонения:
$$\gamma = k \cdot \text{std}(g)$$

### 2. Адаптивность к волатильности
В трейдинге градиенты могут резко менять масштаб при смене рыночного режима. 
Адаптивный TGC (ATGC) пересчитывает $\gamma$ для каждого слоя или каждой итерации:
- **Phase A (Low Variance)**: Увеличиваем $\gamma$ $\rightarrow$ Передаем меньше данных (Sparsity $\uparrow$).
- **Phase B (High Variance)**: Уменьшаем $\gamma$ $\rightarrow$ Передаем больше деталей для быстрой адаптации к рынку.

## Преимущества для торговых систем

1. **Регуляризация**: Обнуление малых градиентов работает как L1-регуляризация, отсекая случайные рыночные флуктуации и предотвращая переобучение на шуме.
2. **Экономия трафика**: При кодировании 2 битами на элемент (для $\{-1, 0, +1\}$) и разреженности 95%+ объем данных снижается в 50-100 раз.
3. **Стабильность**: Сочетание с **Error Feedback** позволяет сохранять информацию об отсеченных градиентах и применять её в будущих итерациях, что критично для захвата долгосрочных трендов.

## Формула Error Feedback в TGC
1. $v_t = \eta g_t + e_{t-1}$ (Добавление ошибки)
2. $u_t = Q(v_t; \gamma_t)$ (Квантование с адаптивным порогом)
3. $e_t = v_t - u_t$ (Сохранение остатка)
4. Передача сжатого вектора $u_t$.
